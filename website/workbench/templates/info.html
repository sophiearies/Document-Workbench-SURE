Welcome to the Document Review Workbench. This information page will tell you everything you need to know about the system. Before getting started, it's important to note that currently this only supports single-user reviewing as of writing this.

Firstly, navigate to the "Review" page. If you have never used this application before, you will not see any reviews to navigate into. Click "New review" to assign your review a title, description and optionally a button to click if you'd like to see a history of actions you've taken when interacting with the review. Once you have created the review, simply click into it to navigate to the actual workbench.

Uploading a .ris file

To get started, simply upload a .ris file to the workbench by clicking "Select Documents". Once you've selected your file, click upload and this will process for a little while. Once this has successfully uploaded, the page should refresh and you should see the total number of documents in the "Review Progress" button.

Annotating the Documents

Clicking "Annotate Documents" will take you to the abstract and title of a document. Then, you will be prompted to assign the document a relevancy. Once you have annotated a good chunk of documents, the "Recall" and "Classify" parts of the application will be more functional.

Review Progress

The next few sections will explain different aspects of the workbench. The review progress contains a list of all everything that's left to do. It's quite intuitive but the total documents is the number that has been uploaded. Relevant/not relevant/undecided are the terms for documents that have been screened in each category. Unexamined is a list of documents that are left to go.

Saving / Exporting Documents

Once you have uploaded your .ris file, you can save the document ID, title and relevancy as either a .csv file or a .txt file. Optionally, if you don't want to export every single document, on the right-hand side of the workbench titled "Export for Review" you can apply different filters on the export. For example, once the dropdown menu is clicked, and "Relevant" is selected, only relevant documents will be exported. The percentage is the total amount of documents out of all that have been decided for a certain category to be exported. You can select the desired file type(either .ris, .txt or .csv).

Classify

The workbench supports priority screening. When the classify button is clicked, the order that the documents are shown in the "Annotate Documents" section will be rearranged. Instead of showing in the order that the document is within the .ris file, documents that are more likely to be relevant are displayed.

Document List

The document list page will send you to a table displaying the document title, relevancy and other information. In the case that you want to change a relevancy, simply click into a document and change the relevancy. Searching for specific document titles here is also supported.

In the case that you want to see past actions that your account has performed, this is visible within the left hand table on the "History" page. Alternatively, if you have enabled history for your review and have selected it on the right hand table, this will display a list of previously performed actions on that review.

On the settings page, there is an option to change the classifier that the program uses. The default classifier to use if no classifier has been selected is Support Vector Machines. The classifier shouldn't make too much of a difference in terms of priority screening. Other than this, the user can also change their role, description, banner picture and profile picture.
